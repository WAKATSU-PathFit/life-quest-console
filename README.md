# Life Quest Console

若林勝則の生活・仕事・お金・集中を見える化するRPG型セルフマネジメントUIプロトタイプです。

## 使用技術

- HTML
- CSS

JavaScriptは現時点では未使用です。

## 公開方法

GitHub Pagesで静的サイトとして公開します。

## GitHub Pages設定手順

1. GitHubでこのRepositoryを開く
2. Settings > Pages を開く
3. Source を Deploy from a branch にする
4. Branch を main にする
5. Folder を /root にする
6. Saveする
7. 表示されたURLをスマホで開く

## 注意

このUIはプロトタイプです。
本物の財布残高、負債、個人情報は入力しないでください。
